# Agent Instructions - iaccess-platform

## Project Context
- **Purpose**: Multi-page HTML platform
- **Type**: Cloudflare Workers + GitHub Pages
- **Status**: Active

## Deployment
- **Cloudflare Dev**: https://iaccess-platform-dev.meauxbility.workers.dev
- **Cloudflare Prod**: https://iaccess-platform.meauxbility.workers.dev
- **GitHub Pages**: https://SamPrimeaux.github.io/iaccess-platform/

## File Structure
- `public/` - HTML files (add/update here)
- `src/index.ts` - Worker with routing (auto-generated)
- `wrangler.toml` - Cloudflare config
- `deploy.sh` - Deploy to Cloudflare
- `deploy-to-github-pages.sh` - Deploy to GitHub Pages

## Update Instructions
When user provides new HTML:
1. Add/update file in `public/` directory
2. Run `./update-worker.sh` (regenerates routing)
3. Preview: `./deploy-to-github-pages.sh`
4. Deploy: `./deploy.sh dev` or `./deploy.sh production`

## Pages
- `iaccess-dashboard.html` → `/`
- `iaccess-ai-gateway.html` → `/iaccess-ai-gateway`
- `iaccess-browser-rendering.html` → `/iaccess-browser-rendering`
